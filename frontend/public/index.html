ℹ️ About Humor Memory Game

🎮 What is this game?

A fun memory card matching game featuring hilarious emojis and jokes! Perfect for testing your memory skills while having a good laugh.

🎯 How to Play

  1. 🎪 Enter your username and select difficulty
  2. 🃏 Click cards to flip them and find matching pairs
  3. ⚡ Match cards quickly for bonus points!
  4. 🏆 Complete all pairs to finish the game
  5. 📊 Check the leaderboard to see your ranking!

🔥 Scoring System

  • 🎯 Base points per match (varies by difficulty)
  • ⚡ Speed bonus for quick matches
  • 💎 Perfect game bonus (no wrong moves)
  • 🔥 Streak bonus for consecutive matches

🛠️ Built for DevOps Learning

This application demonstrates:

  • 🐳 Docker containerization
  • 🗄️ PostgreSQL database integration
  • ⚡ Redis caching

  • 🔒 Security best practices
